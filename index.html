¿Qué es un Sistema Operativo?

Un sistema operativo (SO) es el software encargado de que los móviles, smart TVs y demás dispositivos electrónicos funcionen correctamente. Es el intermediario entre los usuarios y el hardware (pantalla, teclado, ratón, etc.). Permite interactuar con aplicaciones por línea de comandos (ej. en Linux) o mediante interfaces gráficas, y gestiona el almacenamiento y la memoria del dispositivo.

¿Qué es un Sistema Operativo?

El sistema operativo realiza tareas como: gestión de procesos, gestión de memoria, control de dispositivos, gestión de archivos y seguridad. Ejemplos: Linux, Windows, macOS, Android.

  • Gestión de procesos: crea, programa y finaliza procesos.
  • Gestión de memoria: asigna memoria y maneja intercambio (swap).
  • Gestión de archivos: sistemas de archivos y permisos.
  • Control de dispositivos: drivers y comunicación I/O.

Tipos de sistemas operativos

Dependiendo del dispositivo y su funcionalidad, distinguimos varios tipos: sistemas operativos de escritorio, móviles, servidores y embebidos.

Sistemas operativos de escritorio

Desarrollados para portátiles y ordenadores de escritorio. Ejemplos: Windows, macOS, Linux.

Sistemas operativos móviles

Tabletas y teléfonos usan Android, iOS, y otros sistemas móviles.

Sistemas operativos de servidor

Optimizados para uso remoto y manejo de múltiples usuarios simultáneos.

Sistemas embebidos

Programados para funciones concretas: routers, smart TVs, electrodomésticos.

¿Cuáles son las funciones de un sistema operativo?

Entre las principales funciones de un sistema operativo destacan la gestión de recursos, control del rendimiento y realización de copias de seguridad. A continuación un listado más completo:

  • Asignar recursos: Gestiona CPU, memoria y dispositivos.
  • Optimizar la memoria: Administración eficiente del uso de memoria.
  • Ejecutar procesos: Iniciar, planificar y detener procesos.
  • Controlar recursos: Gestiona tiempos de ejecución y acceso a hardware.
  • Garantizar la seguridad: Cifrado, permisos y políticas de acceso.
  • Gestionar hardware: Comunicación con impresoras, ratón, teclado mediante drivers.
  • Administrar conexiones de red: Protocolos y compartición de recursos.
  • Organizar archivos: Crear, organizar y eliminar archivos y directorios.
  • Copias de seguridad: Crear y recuperar respaldos.
  • Llamadas al sistema: Interfaz para que las aplicaciones accedan a recursos del sistema.
  • Máquinas virtuales: Permite ejecutar otros SOs dentro de un equipo mediante virtualización.

¿Qué es el Kernel?

El kernel es el componente central del sistema operativo. Controla el acceso al hardware, gestiona procesos y memoria, y provee primitivas para la comunicación entre procesos.

Funciones clave del kernel

  1. Planificación de CPU y multitarea.
  2. Gestión de memoria (MMU, paginación).
  3. Abstracción de dispositivos (drivers).
  4. Sistemas de archivos y llamadas al sistema (syscalls).

Tipos de kernels

Kernel Monolítico

Toda la funcionalidad del kernel se ejecuta en modo kernel. Ejemplo: Linux.

Microkernel

Minimiza el código en modo kernel, moviendo servicios a espacio de usuario. Ejemplos: Minix, QNX.

Híbrido

Combina ideas de ambos enfoques. Ejemplos: algunos diseños de Windows y macOS.

  • Microsoft Windows NT
  • XNU (macOS)
  • DragonFlyBSD
  • ReactOS
